# ワンピースカード デッキビルダー

ワンピースカードゲームのデッキ作成・カード検索ツールです。

## 機能

### 🔍 カード検索モード
- 色、タイプ、コスト、カウンター、属性、ブロックアイコン、特徴、シリーズでフィルタリング
- フリーワード検索（カード名・テキスト・特徴など）
- 通常カード/パラレルカードの切り替え表示
- 表示列数の変更

### 🧱 デッキ作成モード
- リーダー選択
- カード追加（リーダーの色に合ったカードのみ表示）
- デッキプレビュー
- カード枚数管理（4枚制限、無制限カード対応）
- デッキエクスポート（テキスト形式）
- デッキインポート（テキスト/QRコード）
- デッキ画像生成（リーダー + カード50枚グリッド + QRコード）

## 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **画像処理**: Sharp
- **QRコード**: qrcode, jsqr

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# 本番サーバーの起動
npm start
```

## デプロイ

Vercelへのデプロイを推奨します。

1. GitHubにリポジトリをプッシュ
2. Vercelでプロジェクトをインポート
3. 自動でビルド・デプロイされます

## データファイル

`data/` ディレクトリに以下のCSVファイルを配置してください：

- `cardlist_filtered.csv` - メインカードリスト
- `cardlist_p_only.csv` - パラレルカードリスト
- `custom_cards.csv` - カスタムカードリスト（オプション）
